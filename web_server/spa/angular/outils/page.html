Outils utiles


Cette page recense un ensemble de liens utiles pour gérer la sécurité de sa maison.

Virus Total
Site web qui analyse les fichiers suspects et facilite la détection rapide des virus, vers, chevaux de Troie et toutes sortes de logiciels malveillants (« malwares » en anglais) détectés par les moteurs antivirus.

HaveIbeenpwned
Site Web qui permet aux utilisateurs de vérifier si leurs données personnelles ont été compromises.

Avast
Antivirus propriétaire comportant une version gratuite pour une utilisation personnelle et non commerciale.
